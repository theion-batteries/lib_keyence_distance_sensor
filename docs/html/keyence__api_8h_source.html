28 /*
29 * the commands must be sent to the device using CR as delimeter,
30 * and the response must be received also with CR.
31 * the controller (arduino) must send one command and wait for
32 * the response to send next one.
33 * General Mode: During measurement The measurement control commands are accepted.
34 Commands such as writing/reading setting values are not accepted
35 * Communication Mode: • When the command "Q" "0" "CR" is received, the mode changes to
36 the communication mode, and the setting values are written/read.
37 • The measurement stops during the communication mode.
38 * command | description | Mode | response | ERROR
39 --------------------------------------------------------------------------
40 * Q0 CR | Changing to the communication mode | communication | same as command | Err-51
41 --------------------------------------------------------------------------
42 * R0 CR | Changing to the general mode | general | same as command | ERROR
43 --------------------------------------------------------------------------
44 * MS,01 CR | Measured value output (single) 01-12 | General | MS,OUT01,value CR | ERROR
45 --------------------------------------------------------------------------
46 --------------------------------------------------------------------------
47 * MM,010010000000 CR | Measured value output (multiple) 01-12 ex here out2 and out5| General | MM,010010000000,value[,value,value] CR | ERROR
48 --------------------------------------------------------------------------
49 * MA CR | Measured value output (all) 01-12| General | MA,value[,value,value] CR | ERROR
50 --------------------------------------------------------------------------
51 * Timing Diagrams
52 * t: time between receiving command and responding: process time
53 * Mesrument/control PW: 100ms + number of heads x 100ms : if one head: 200ms if 3 heads 100ms +3*100=400ms
54 * Mode Change to General: 600 ms + Number of head expansion units x 750 ms = 600+3*750 = 2850 ms
55 * Error Code
56 • 50: Command error
57 • 51: Status error
58 • 60: Command length error
59 • 61: Parameter count error
60 • 62: Parameter range error
61 • 88: Timeout error
62 • 99: Other error
63 */
